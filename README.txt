Installation
-------------
This application runs in all current browsers and IE9 and above.

In order for it to run off a local build on ie9, you must enable "Access Data sources across domains." To do this, go to Settings->Internet Options->Security->Custom Level and change security settings under "Miscellaneous" set "Access data sources across domains" to Enable.

Controls
---------------
Click on thumbnails to view them as a lightbox. Click the left or right to navigate left or right. Also use the keys left and right to navigate as well. Escape or clicking the "X" will close the lightbox.s
